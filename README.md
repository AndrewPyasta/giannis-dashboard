# Giannis Antetokounmpo Career Dashboard (2014–present)
This project is an interactive Power BI dashboard that analyzes Giannis Antetokounmpo’s NBA career from 2014 to 2025. 
It highlights career totals, season trends, efficiency metrics, and award history using advanced basketball statistics.

## Dashboard Features

### Career Overview
- Total career points, assists, and rebounds
- Career averages per season
- Award history including MVPs, championships, and All-NBA selections

### Season Trends
- Points by season line chart
- Field goals made by season bar chart
- Top scoring seasons comparison

### Shooting Breakdown
- Field goals made
- Three-pointers made
- Free throws made

### Interactive Elements
- Tooltips with additional stat context
- Filters by season
- Clean KPI cards for quick insights

## Tech Stack
- Power BI Desktop
- Power Query (data cleaning & transformation)
- DAX (measures and KPIs)
